# 👨🏻‍💻 Customer Behavior Analysis Project

This project is an end-to-end data analytics project where I worked on transforming raw customer data into meaningful business insights. The goal was to understand customer behavior, purchasing patterns, and provide actionable recommendations using data.

---

## 📌 Project Objective

The main objective of this project is to:

- Analyze customer shopping behavior  
- Identify different customer segments  
- Understand factors influencing purchases  
- Present insights using dashboards and reports  

This project helped me strengthen my skills in **Python, SQL, and Power BI** while working on a real-world dataset.

---

## 🧠 What I Learned

Through this project, I gained hands-on experience in:

- Data cleaning and preprocessing  
- Writing SQL queries for business analysis  
- Performing Exploratory Data Analysis (EDA)  
- Creating interactive dashboards  
- Converting data insights into business recommendations  

---

## 🔧 Tools & Technologies Used

- **Python** (Pandas, NumPy, Matplotlib, Seaborn)  
- **SQL** (Joins, Aggregations, CTEs)  
- **Power BI** (Dashboard creation & visualization)  

---

## 🔄 Project Workflow

### 1. Data Cleaning & Preparation (Python)
- Handled missing values  
- Removed duplicates  
- Converted data types  
- Prepared dataset for analysis  

---

### 2. Exploratory Data Analysis (EDA)
- Identified trends and patterns  
- Analyzed customer demographics  
- Studied purchase behavior  

---

### 3. Data Analysis (SQL)
- Extracted insights using SQL queries  
- Performed customer segmentation  
- Analyzed revenue contribution  
- Identified high-value customers  

---

### 4. Dashboard Creation (Power BI)
- Built an interactive dashboard  
- Visualized key metrics:
  - Sales trends  
  - Customer segments  
  - Product performance  

---

### 5. Insights & Recommendations
- Identified loyal vs at-risk customers  
- Suggested strategies for customer retention  
- Highlighted key revenue drivers
